# test-monitor

## Запуск

```bash
sudo chmod +x ./install.sh
sudo ./install.sh
```

Этот скрипт скопирует юниты в /etc/systemd/system/ и сам скрипт в /usr/local/bin/, а также активирует юниты

## Скрипт

Программа сравнивает старый pid процесса из файла и текущий pid test. Если не совпадают, логирует. Отправляет запрос на указанный url, однако сервер никогда не отвечает(
